### 🚀 Featured Project

**TermaSSH** — My recent open-source project

[![TermaSSH](https://github.com/beliberda/TermaSSH/blob/main/public/termassh-screenshot.png?raw=true)](https://github.com/beliberda/TermaSSH)

**TermaSSH** is a **free, open-source desktop SSH client + SFTP/FTP file manager** — a lightweight alternative to MobaXterm.

**Key features:**
- Multi-tab SSH terminal (powered by xterm.js)
- Built-in double-panel SFTP/FTP file browser
- Session manager with import/export
- Support for password & SSH key authentication
- Dark/Light themes, keyboard shortcuts, i18n (EN/RU/DE/IT/ZH)
- Built with **Tauri 2 + Rust + React + MobX**

→ [GitHub Repository](https://github.com/beliberda/TermaSSH)  
→ [Releases (Windows / macOS / Linux)](https://github.com/beliberda/TermaSSH/releases)
